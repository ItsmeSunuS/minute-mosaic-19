Imagine having a tool that helps you gain insights into how you spend your day.The Track Your Time With Me is a modern web application designed to help users track daily activities and visualize time usage with ease and allows them to log their daily activities (e.g., Work, Study, Sleep, Exercise) along with track time spent on each.Users can create an account, log in securely, and record their daily activities along with the time spent on each one. The system ensures that the total time recorded does not exceed 24 hours (1440 minutes).All data is stored in a Firebase Realtime Database using REST API calls, making the app lightweight and easy to deploy.

Key features include:
* User authentication via Firebase (Email/Password or Google login)
* Activity logging with categories and duration (up to 1440 minutes per day)
* Data visualization via interactive pie charts
* Responsive Ui for both desktop and mobile devices
* Data is stored securely in Firebase Realtime Database using REST API calls, ensuring seamless and lightweight operation.
  This tool is ideal for users who want to optimize their time management or simply understand where their hours are going each day.

## Features

Feature 1: User Authentication with Firebase Auth
Description: The app offers secure user authentication using Firebase Auth, allowing users to sign up and log in with Email/Password or Google login, ensuring that only authorized users can access their time-tracking data.

Task Analysis

Description: The app analyzes how users spend their time by categorizing and tracking activities throughout the day, giving users a clearer understanding of how their time is allocated across different tasks (e.g., Work, Study, Sleep).
---
Feature 3: AI-Powered Task Suggestions
Description: The app leverages AI algorithms to provide intelligent suggestions for tasks based on users' past activity patterns, helping them optimize their schedules and improve productivity.

Feature 4: Real-Time Data Sync with Firebase Realtime Database
Description: The app uses Firebase Realtime Database to store and sync activity data in real time, ensuring that users' logs are always up-to-date and accessible from any device without delay.
